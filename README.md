A tool for labeling foreground and background in images.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/my-image-labeler.git
   cd my-image-labeler
Install the required dependencies:
bash
pip install -r requirements.txt
Usage
Run the labeling script:

bash
python labeler.py
Follow the instructions to load an image, label foreground and background, and save the labels.

## Contributing
We welcome contributions! Here's how you can help:

## Fork the repository.
Create a new branch with your feature or bugfix.
Submit a pull request.
Uploading Your Labeled Data
Save your labeled image and metadata in the user_uploads/ directory.

## Make sure to include:

The labeled image.
The label file (e.g., image_labels.txt).
A prompt file (prompt.txt) with a short description of what you labeled (e.g., marking foreground for the cherry tree).
Submit a pull request with your uploads.
